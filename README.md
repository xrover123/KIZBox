Сбор кизов и количеств по каждому из КИЗов для использования в пользовательских приложениях Паруса.

Сканирование происходит в верхние текстовое поле. Если в сканере включено добавление перевода строки, то после сканирования КИЗ добавляется автоматически в список, расположенный ниже.
Если один и тот же КИЗ добавляется несколько раз каждый раз количество по данному кизу (поле внизу интерфейса) увеличивается на единицу.
После закрытия программы в каталоге запуска формируются файлы "KIZ.lst" и "CNT.lst" если нажата кномка OK. В любом случае формируется файл "EOF.stp"
